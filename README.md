Download links
- Windows x64: https://downloader.sfbot.eu/updates/sfbot.exe
- Windows Installer: https://downloader.sfbot.eu/updates/sfbot_installer.exe
- Linux x64: wget https://downloader.sfbot.eu/updates/sfbot-linux-x64
- Linux ARM64 (armv8/aarch64): wget https://downloader.sfbot.eu/updates/sfbot-linux-arm64
- Linux ARMv7 (32-bit): wget https://downloader.sfbot.eu/updates/sfbot-linux-armv7
- Linux i686 (32-bit x86): wget https://downloader.sfbot.eu/updates/sfbot-linux-i686
- latest.json (versions/info): https://downloader.sfbot.eu/updates/latest.json

Usage
- Windows: start sfbot.exe. The tray icon appears, right click -> "Open UI".
- Linux: run ./sfbot (if needed `chmod +x sfbot`). Then open http://localhost:3000 in your browser.
